# UNDERSTANDING THE FUNDAMENTALS OF A GAME

### GAME LOOP

1. Process Input
* Some external action:

	- Button Press

	- Mouse Click

2. Update Game
* Chaging things on the screen

	- Collisions
	
	- Time running
	
	- Character movement

3. Render
* Drawing what happened in step 2


### Drawing

Double buffering:

Drawing       ->   Display
(back side)  Flip  (Front side)
